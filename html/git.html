Git

  • git init - Инициализирует пустой репозиторий Git в текущей директории.
  • git clone - Копирует удаленный репозиторий на локальный компьютер.
  • git add - Добавляет изменения файлов в индекс (стадия подготовленных изменений перед коммитом).
  • git commit - Создает снимок всех внесенных изменений и фиксирует их в истории проекта.
  • git push - Отправляет локальные коммиты на удалённый сервер (например, GitHub, Bitbucket).
  • git pull - Загружает новые изменения с удалённого сервера и объединяет их с вашей рабочей веткой.
  • git branch - Управляет ветвями: создание новых ветвей, переключение между ними, просмотр существующих ветвей.
  • git checkout - Переключается между разными версиями (ветвями) проекта или отдельными файлами.
  • git merge - Объединяет две ветви вместе, интегрируя изменения одной ветки в другую.
  • git log - Показывает историю предыдущих коммитов, включая автора, сообщение и метку времени каждого коммита.
